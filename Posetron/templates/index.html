Our Functionality

PoseTron:

PoseTron algorithm

PoseTron feature concept is a implementation of the Posenet algorithm for detecting inappropriate poses in the workpace.

PoseTron usage

We think it can be implemented along with CCTV cameras to track the body language of office workers and alert staff of misconduct.

PoseTron effectiveness

Since this model is trained on the Movenet algorithm by tensorflow, it can detect inappropriate static poses really well and raise an alarm

Converserator:

Converserator algorithm

Converserator is a employee training tool which makes use of Natural Language Processing to effectively train employees. It uses the language layer API to process the textual language and the _ to convert speech to text.

Converserator Usage

It can be implemented as an employee training tool and help promote inclusivity by prompting the employee to speak in the language everyone understands if he speaks a language someone does not understand.

Converserator Effectiveness

This feature is still under development and testing. Since this feature uses Neural Networks for obtaining the language spoken by the user, slight errors can be seen. However, we believe it can be a powerful tool once development is finished.
